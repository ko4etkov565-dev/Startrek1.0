Description:
This game was developed based on a tutorial and significantly expanded with custom features and visual enhancements.
Key Enhancements:
Visual Effects: Implemented a meteorite rain effect using the Unity Particle System.
Environment: Added a dynamic background with a Parallax scrolling system.
Camera & Cinematics: Configured Cinemachine cameras with a smooth zoom-in effect at the start of each level.
Animations: Created animations for coins, fuel tanks, and the launcher using Animator and Timeline.
Content: Designed and added 5 unique levels, transforming the tutorial base into a complete game experience.
